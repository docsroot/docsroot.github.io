1/ Meta tags overview

These tags are inside the <head> tag and are optional descriptors for our page. They will help to tell users and search engines what is about our document even there is some debate on which tags are usefull for search engines and they are not a magic way to boost the rank on SE.

2/ Title tag

This is a required tag of a HTML document. It will display in browser's title and also visible in SE results.

4/ Keywords Meta Tag

The early SEs took this tag into consideration but not now. So we shouldn't use it even not needed to spent time to remove it. But it will help to reduce page size.

5/ Robots Meta Tag

This is very important meta tag

  • CONTENT="NOINDEX, NOFOLLOW": means that don't index this page, don't follow links from this page
  • CONTENT="INDEX, NOFOLLOW": means that do index this page, don't follow links from this page
  • CONTENT="NOINDEX, FOLLOW": means that don't index this page, do follow links from this page
  • CONTENT="INDEX, FOLLOW": means that do index this page, do follow links from this page

6/ Charset Tag

This will help browser to view your page in correct character set
