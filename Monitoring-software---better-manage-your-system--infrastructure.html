Monitoring software - better manage your system & infrastructure

What a monitoring tool will help us?

* To see what devices / components are available

* To see how busy / usaged a server is being (80% of CPU), free disk spaces

And we can also use a monitoring software to optimize / save cost of investment, eg:

* A server that only is being used 30% of CPU

* A server that never exceeds 50% of RAM...

We can decide to reduce the load of resources especially in a cloud platform that we pay by what we build (CPU / RAM / Disk...)
