DocsRoot - What components are used to build the web app

If you would like to find components to build a web app or want to check out what are used to build the docsroot app, this page is for you.

DocsRoot is a web based application based on Asp.net MVC to provide an easier way to manage your documents. It is built based on below components:

Server

Windows server with IIS

SQL Server

Framework

ASP.NET MVC version 5

Bootstrap v3 with jQuery

Libraries

Bootboxjs for Modals

reCaptcha for spam anticipant
