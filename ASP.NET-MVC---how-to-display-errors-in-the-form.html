ASP.NET MVC - how to display errors in the form

In every application, there will have a form to create some entities or a form that let users submit their feedback.

The data input is not always valid and validation is a required step before saving data into storage (database / filesystem...).

ASP.NET MVC supports a nice way to validate all form input and capture them and print all errors on the form.

1/ Decorating your form's view model with attributes

This link Validation with the Data Annotation Validators provide a view on how validation works.

As an example, I have this view model with a simple requirement: the fields must have values.

    public class EditDocumentViewModel
    {
        [Required]
        public int Id { get; set; }
        [Required]
        [AllowHtml]
        public string Body { get; set; }
        public string Title { get; set; }
        public string FileName { get; set; }
        public string Description { get; set; }
        public string Author { get; set; }
    }

2/ Logic in Controller

We can perform additional validation on controller's action then add errors into ModelState


3/ Display all errors on the form

Use below method to view all errors from ModelState

@Html.ValidationSummary(false)
