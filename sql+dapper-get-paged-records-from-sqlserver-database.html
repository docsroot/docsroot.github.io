In this page you will find the steps to query for just a page of records from database by using an SQL proc and execute it via Dapper extension method

Step 1: Create a sproc

Below is a sample sproc, you must change the ORDER BY and table name with yours

create proc GetDocumentsWithPagination (
@SkipOffset integer,
@TakeNumber integer,
@Total integer output
)
as
begin
select @Total = COUNT(*) from dbo.Documents

;with CTE as (
select *, ROW_NUMBER() OVER(ORDER BY Title) as RowNumber
from dbo.Documents
)
select * from CTE
where RowNumber > @SkipOffset and RowNumber < @SkipOffset + @TakeNumber
end

Step 2: Write C# code to execute the sproc

Below C# code which performs

* Build three parameters as a dynamic dictionary

* Note on Total param, it has dbType and direction

* Just as usual way with Dapper to get the list of records; use parameters.Get method for Total records.


            using (var connection = CreateDbConnection(_connectionString))
            {
                var parameters = new DynamicParameters();
                parameters.Add("SkipOffset", page.GetOffset());
                parameters.Add("TakeNumber", page.ItemsPerPage);
                parameters.Add("Total", dbType:DbType.Int32,  direction: ParameterDirection.Output);
                var result = connection.Query<DocumentInfoEntity>("[dbo].[GetDocumentsWithPagination]",
                    commandType: CommandType.StoredProcedure, param: parameters);
                var total = parameters.Get<int>("Total");
                return new Result<DocumentInfoEntity>(result)
                {
                    Total = total
                };
            }
