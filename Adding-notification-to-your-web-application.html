Adding notification to your web application

It's pretty common in web apps that need to have a notification after user's action.

Below is the option I apply into my asp.net mvc website: toastr

1/ Download css & js files

Go to the cdn website and take latest files into Content / Scripts folder (for debugging)

2/ Adding new bundles (css & javascript)

bundles.Add(new ScriptBundle("~/bundles/toastr", "//cdnjs.cloudflare.com/ajax/libs/toastr.js/2.1.4/toastr.min.js")
        .Include("~/Scripts/toastr.min.js"));
    bundles.Add(new StyleBundle("~/Content/toastr", "//cdnjs.cloudflare.com/ajax/libs/toastr.js/2.1.4/toastr.min.css")
        .Include("~/Scripts/toastr.min.css"));

3/ Render the bundles into expected view
