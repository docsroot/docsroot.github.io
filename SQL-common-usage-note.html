1/ Order on column with nullable value

If a column has integer value to be used for ordering, null rows are being smaller than 0.

Below is an example, the table has 3 rows only. When "order by [Order] desc", the result will be:

When we want it (null) to appear at the end of list, use a case when / isnull clause like

order by ISNULL([Order], 1000) desc  


order by
case when [Order] IS NULL then 10000
else [Order] 
end
