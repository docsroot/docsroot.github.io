Good options to create an action confirmation

In application development we will have actions that can delete data, send important information to server/other services/clients. As a user, he may click on that kind of actions by mistake or he still has more thing to update or wait for better time... there are several reasons to undo / postpone the action. That's why a confirmation should be display...


How to implement this in a right way or can we reuse it / save time / avoid any issue ? From here I'll list some options.

Option 1: Go with bootstrap

If your project already uses bootstrap, one good js lib you could checkout is bootbox - it's designed to make the use of Bootstrap Models easier.

Here are all steps to archive the goal:

This example is pure HTML

Sample is with asp.net mvc

Step 0 - Add references (via bundles / scripts)

Step 1 - Add your html tags with a role you want the box will be happened (eg: data-role="delete")

Step 2 - Add some js code to your page like the below - which will display a confirmation for all the links with data-role is delete:

        // Display confirmation before delete
        $('a[data-role=delete]').on('click', function (e) {
            if (!bootbox) {
                return;
            }
            e.preventDefault();
            var href = $(this).attr('href');
            bootbox.confirm("Are you sure you want to delete?", function (result) {
                if (result === true) {
                    location.href = href;
                }
            });
        });
