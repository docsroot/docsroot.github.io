Cloud notes - AWS / GCP

1/ AWS - Elastic IP (https://aws.amazon.com/ec2/pricing/on-demand/#Elastic_IP_Addresses)

When an instance does not have static IP, it will be assigned a random IP from the pool on each start.

We can reserve an IP with very low cost, it will be charged only if the instance mapped to it is off. So the price can be 0.005$ / hour (off).
