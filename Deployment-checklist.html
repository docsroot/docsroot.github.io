Deployment checklist

All steps below are required to be done for each deployment

1. Take a backup of databases by using DbBackupMaster

2. Update Trello the information of the new release (version to be installed, date)

3. Start the installation with Jenkins and check until there is no issue

4. Test the application to be sure that it works fine

5. Copy the database backup files into revelent folders in AllApplicationReleases repository

6. Copy the new releases into AllApplicationReleases

7. Commit and Push the new files, to make sure that they are available in BitBucket.
