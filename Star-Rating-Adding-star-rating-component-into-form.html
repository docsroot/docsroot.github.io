Star-Rating: Adding star rating component into form

I have a review form in my application that represents the value that customers want to set for their satifaction. But if using a textbox with value from 1...5 is not so good, then a star-rating is asked to be added.

After searching for a open source component, I come up with Star-Rating which bases on Bootstrap already. Its usage is also very easy with some simple steps. Below I share what was done for my ASPNET MVC project

1/ Define two bundles for JS and CSS

    bundles.Add(new StyleBundle("~/Content/star-rating", "//cdnjs.cloudflare.com/ajax/libs/bootstrap-star-rating/4.0.5/css/star-rating.min.css")
        .Include("~/Content/star-rating.min.css"));
    bundles.Add(new ScriptBundle("~/bundles/star-rating", "//cdnjs.cloudflare.com/ajax/libs/bootstrap-star-rating/4.0.5/js/star-rating.min.js")
        .Include("~/Scripts/star-rating.min.js"));
2/ Adding the bundles into my view page cshtml and activate the input element that I want

@Scripts.Render("~/bundles/star-rating")
@Styles.Render("~/Content/star-rating")
<script type="text/javascript">
    $("input#Point").rating({ min: 0, max: 5, step: 1, size: 'md', showClear: false, showCaption: true });
</script>
3/ Finally watch for the result
