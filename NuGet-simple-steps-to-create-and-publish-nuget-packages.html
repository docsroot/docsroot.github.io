NuGet simple steps to create and publish nuget packages

When we want to publish some DLL files as a package onto nuget.org, it is required to create a nupack file then publish it.

I was using NuGet Packager (it creates a project, add dll files into libs folder, update nuspec files, build the project in release mode, then a package will be published on nuget.org). But it is not being maintained and seems being outdate on new nuget version.

Below are simple steps to build/publish a package:

1/ Create a Package.nuspec file

Below is a sample (replace id, version, title, owners, dependencies... by yours).

Then besure that all files are added into folders libs / content / tools.

<?xml version="1.0"?>
<package >
  <metadata>
    <id>Rabbit.Web.MvcNotification</id>
    <version>1.2.0</version>
    <title>Rabbit.Web.MvcNotification</title>
    <authors>juniho</authors>
    <owners>RabbitSoftware</owners>
    <description>
      Provides an option to display user messages
    </description>
    <releaseNotes>
    </releaseNotes>
    <summary>
    </summary>
    <language>en-US</language>
    <projectUrl>https://nuget.org/packages/NuGet.Rabbit.Web.MvcNotification</projectUrl>
    <iconUrl>https://nuget.org/Content/Images/packageDefaultIcon-50x50.png</iconUrl>
    <requireLicenseAcceptance>false</requireLicenseAcceptance>
    <licenseUrl>http://opensource.org/licenses/Apache-2.0</licenseUrl>
    <copyright>Copyright @ RabbitSoftware 2018</copyright>
    <dependencies>
        <group targetFramework="net46">
          <dependency id="Microsoft.AspNet.Mvc" version="5.2.6" />
        </group>
    </dependencies>
    <references></references>
    <tags></tags>
  </metadata>
  <files>
    <file src="lib\" target="lib" />
    <file src="tools\" target="tools" />
    <file src="content\" target="content" />
  </files>
</package>

2/ Create a package with nuget pack

The command to use is

nuget pack <Path To Package NuSpec File>

3/ Publish the package to nuget.org

The command below will do the job

nuget push <Path To nupkg File> -Source https://api.nuget.org/v3/index.json
