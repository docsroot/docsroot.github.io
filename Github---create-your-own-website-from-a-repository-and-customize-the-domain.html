Github Pages has a great feature: it allows you to host your static (html/css/javascript) website without any change.

It also supports to add your custom dYomain over its default sub domain (like docsroot.github.io and your custom domain is docgit.devcovery.com).

Below are few easy steps to make everything work:

1/ Create your own account and an organization

After having the org, create a repository with this format: <organization name>.github.io, example: docsroot.github.io


2/ Create sample content and push it to git repository. You can use online editor to craete new files, eg hello-docsroot.html

Now your site is available at http://<organization name>.github.io, eg docsroot.github.io

3/ Add a custom domain if you already have it (like doit.devcovery.com)
