Các phần mềm cho PDF

Dưới đây là một số phần mềm / app cho việc chỉnh sửa, tạo file PDF

1/ Sửa file PDF online - smallpdf

https://smallpdf.com/edit-pdf

Sau khi upload file PDF lên là có thể sửa, theo các menu như sau.

Sửa xong click nút Apply ở góc dưới bên phải. Và tải file về theo các tùy chọn như này:


2/ Sửa file PDF online và free - pdfescape. Lợi thế - nhiều lựa chọn hơn thay vì phải tải file lên từ máy của bạn.

https://www.pdfescape.com/open/

Với các lựa chọn như sau, giả sửa chọn "Upload..." để tải file lên.

Sau đó chọn Trang muốn sửa, và các menu ở góc trên bên trái

Sửa xong thì lựa chọn Lưu hoặc Tải xuống - menu bên trái phía trên.
